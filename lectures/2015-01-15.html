More DAL options

Added to individual fields. Ex. db.mydatabase.field1

  • .label define a schema (how DB fields appear to the programmer)
  • .default define DB values until explicitly defined
  • .requires takes a validation function to validate a field
  • .readable if true, never shown in a view
  • .writable set to true or false defines whether a DB entry can be changed after initialization

request

A container structure with all the variables from the user request. Access with request.var or request.args(0)

Redirect

redirect(URL('default','index')

session.flash

A built-in queue for alert messages. Standard use to append to queue is:
session.flash = T("message")

Nicer SQLFORM

A more informative form, showing specified entries and having edit/delete functionality 

form = SQLFORM.grid(q,
fields=[db.mytable.name,db.mytable.email,db.mytable.date
editable=false
deletable=false
)
